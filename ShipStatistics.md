# Renegade Legion: Leviathan — Ship Statistics Reference

All playable and scenario ship classes with key statistics for review.  
*Source: `index.html` — SHIP_CLASSES and ENHANCED_SHIP_CLASSES.*

---

## Legend

| Abbrev | Meaning |
|--------|--------|
| Size | Hull size (affects PD slots, targeting, boarding) |
| Thrust | Movement points per turn |
| Max Speed | Maximum speed (RU) |
| Turn | Turn rate (degrees) |
| Shields | Fore / Aft shield % (block chance, drain on hit) |
| Turrets | Left / Right PD turret count (base) |
| EW | Electronic warfare rating |
| Crew | Total / Quality / Captain |
| Spinal | Spinal mount type and count |
| Broadside | Laser/bay type and total count (L+R) |
| Fore/Aft | Forward/aft battery counts |
| VLS | Vertical launch system: groups × cells or total missiles |
| Fighters | Onboard / total, launch rate, default type |
